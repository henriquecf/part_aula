== PartAula

{<img src="https://magnum.travis-ci.com/henriquecf/part_aula.svg?token=ZgZQ2kWgaytUQDePu3z8&branch=master" alt="Build Status" />}[https://magnum.travis-ci.com/henriquecf/part_aula]


Requirements:

* Rails 4.2

* Ruby >= 2

* Postgres DB >= 9.3

* For the first time, run rake db:setup

* To deploy on heroku, set heroku remote and run git push to it (do not forget to migrate if necessary)
